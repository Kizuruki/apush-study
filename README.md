# APUSH Study App

AP US History study tool with AI-generated questions where every answer choice is historically accurate. Wrong answers describe real events — they're wrong because they address different causes, effects, or connections than what the question asks.

## Features

- **All 9 CED Units** with every topic, content summaries, key terms, and illustrative examples
- **Weighted full review** — questions appear proportional to exam weight (10-17% units appear most)
- **Per-topic progress tracking** saved in browser localStorage
- **Missed questions tracker** — review what you got wrong
- **Essay prompt generator** — LEQ, SAQ, and DBQ angles per unit
- **Fiveable notes** integration — run the extractor script to add rich notes
- **Heimler PDF notes** — drop PDFs in notes/heimler/ and extract them

## Quick Start (GitHub Pages)

1. Fork this repository
2. Go to **Settings → Pages → Source: main branch / root**
3. Visit `https://yourusername.github.io/apush-study/`
4. Enter your [Anthropic API key](https://console.anthropic.com) when prompted
5. Start studying!

## Quick Start (Local)

```bash
# Serve locally (required — file:// won't work for module imports)
python -m http.server 8080
# Then open: http://localhost:8080
```

## Adding Fiveable Notes (Optional)

Adds rich study notes from Fiveable to every topic card.

```bash
pip install openai requests beautifulsoup4

# Set your OpenAI API key
export OPENAI_API_KEY="sk-..."          # Mac/Linux
$env:OPENAI_API_KEY="sk-..."            # Windows PowerShell

python fetch_fiveable.py               # fetch all 66 topics (~$0.13)
python fetch_fiveable.py --resume      # resume if interrupted
```

Output: `src/data/fiveable_notes.json` — commit this to your repo and notes will show in the app.

## Adding Heimler PDF Notes (Optional)

1. Drop Heimler PDF files into `notes/heimler/`
2. Run the extractor:

```bash
pip install openai pdfplumber

export OPENAI_API_KEY="sk-..."

python extract_heimler.py
```

Output: `src/data/heimler_notes.json`

## API Key Security

Your Anthropic API key is stored **only in your browser's localStorage**. It is never sent anywhere except directly to `api.anthropic.com`. The repo contains no API keys. Each user of the deployed site needs to enter their own key.
